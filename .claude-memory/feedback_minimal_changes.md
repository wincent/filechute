---
name: Don't make unnecessary visual changes
description: User objected when search field was moved from toolbar to content area without discussion — prefer minimal changes and explain constraints before making big design shifts
type: feedback
---

Don't move UI elements to different locations without discussing the
trade-off first.

**Why:** When asked to fix tab ordering, the search field was moved from
the toolbar to a content-area filter bar without asking. The user
objected: "why did you completely move the toolbar field? was that
necessary?"

**How to apply:** When the simplest fix would change the visual design,
explain the constraint and ask the user which approach they prefer rather
than silently making a large change.
